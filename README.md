## Features

- **User Authentication**: Secure login and session handling using Passport.js.
- **Budget Management**: Add, edit, and delete budget categories and their respective amounts.
- **Interactive UI**: Dynamic interface for managing budget items, including inline editing.
- **Database Integration**: Persistent storage of user and budget data with MongoDB.
- **Responsive Design**: Mobile-friendly layout using Bootstrap for a seamless experience on any device.

## Skills Learned

This project allowed me to develop and strengthen a variety of technical skills, including:

### Backend Development
- Setting up a **Node.js** and **Express.js** server.
- Integrating **MongoDB** and managing data with **Mongoose**.
- Designing and implementing RESTful routes for **CRUD operations**.

### Authentication and Security
- Implementing authentication with **Passport.js**.
- Managing sessions securely with **express-session**.
- Providing user feedback with **connect-flash**.

### Frontend Development
- Using **EJS** for server-side templating and dynamic HTML rendering.
- Creating responsive layouts with **Bootstrap** and **Font Awesome**.
- Adding client-side interactivity with **vanilla JavaScript**.

### Full-Stack Workflow
- Bridging frontend and backend seamlessly for a cohesive application.
- Debugging across all layers of the stack (client, server, database).
- Following best practices for code organization, modularity, and scalability.
